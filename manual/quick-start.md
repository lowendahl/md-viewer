# Quick Start

## 1. Open a folder

Click **📂 Open folder** in the top-left corner and pick the folder
that contains your Markdown notes.

The tree appears on the left:

- 📁 folders are expandable
- 📄 markdown files (click to open)
- 🖼 images (click to view in lightbox)

## 2. Edit a file

Click a file in the tree. The editor opens in the middle panel.

- Type `### ` and a space → instantly becomes an H3 heading.
- Type `- ` to start a bullet list, `1. ` for numbered.
- Type ` ``` ` (triple backtick) to start a code block.
- Drag any image from your desktop or the tree into the editor.

The save state in the top-right shows **saved** / **unsaved**.
MD Viewer auto-saves 1.5 seconds after you stop typing.

## 3. Add review comments

Right-click any text in the editor and pick **Add comment**. Type
your note. Comments live in a sidecar file next to your `.md` — they
don't pollute the source.

The right rail (📋 Comments) lists all open comments. Click one to
jump to the anchor.

## 4. Search

| Shortcut | What it does |
|---|---|
| **Ctrl+F** | Find in the open file |
| **Ctrl+P** | Quick switcher (jump to file by name) |
| **Ctrl+Shift+F** | Full-text search across the whole folder |
| **Ctrl+L** | Focus the tree filter input |

## 5. Manage files

Right-click anywhere in the tree:

- New file / new folder
- Rename
- Delete
- Open in Explorer

Or use:

- **Drag-and-drop** files between folders.
- **Alt+drag** to drag a file out to another app (e.g. Outlook).
- **Del** key to delete the selected file/folder.
