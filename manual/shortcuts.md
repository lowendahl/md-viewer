# Keyboard Shortcuts

## Files

| Shortcut | Action |
|---|---|
| **Ctrl+S** | Save the current file |
| **Ctrl+P** | Quick switcher |
| **Ctrl+Shift+F** | Full-text search |
| **Ctrl+F** | Find in current file |
| **Ctrl+L** | Focus tree filter |
| **Del** | Delete selected file/folder |

## Editor

| Shortcut | Action |
|---|---|
| **Ctrl+Z** | Undo |
| **Ctrl+Shift+Z** | Redo |
| **Ctrl+B** | Bold |
| **Ctrl+I** | Italic |

## Markdown shortcuts (typed inline)

| Type | Result |
|---|---|
| `# ` | H1 heading |
| `## ` | H2 |
| `### ` | H3 |
| `- ` | Bullet list |
| `1. ` | Numbered list |
| `> ` | Blockquote |
| `[[` | Wikilink autocomplete |

## Images

| Action | How |
|---|---|
| Insert image | Drag/drop from tree or filesystem; or paste from clipboard |
| Open in lightbox | Click image |
| Zoom in/out | Mouse wheel, or − / + buttons |
| Fit / 1:1 | Toolbar buttons |
| Pan | Drag when zoomed |
| Close lightbox | Esc, or click outside the image |

## Tree

| Shortcut | Action |
|---|---|
| Click | Open file |
| Right-click | Context menu (rename, delete, new, open in Explorer) |
| Drag | Move file to another folder |
| **Alt+drag** | Drag file out to another app |
| Click image | Open in lightbox |

## Window / app

| Shortcut | Action |
|---|---|
| **Esc** | Close modal / lightbox / find bar |
