# Sidemark demo

This file ships with a sidecar (`hello.review.yaml`) so you can see Sidemark
comments rendered as gutter badges and inline highlights.

## Try it

- Click a 💬 badge in the right gutter to read a thread.
- Select some text and click the floating ➕ to add a comment.
- Resolve / reply / delete from the popover.
- Toggle the **Comments** checkbox in the topbar to hide/show.

> Comments are stored in a sidecar file — your markdown stays clean.
